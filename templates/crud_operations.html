CRUD Operations

Perform CRUD Operations via Postman

Use the following endpoints:

  • GET Books: /books/list
  • POST Book: /books/add (Body: JSON)
  • POST Book: /books/edit/{id} (Body: JSON)
  • POST Book: /books/delete/{id}
  • GET Members: /members/list
  • POST Member: /members/add (Body: JSON)
  • POST Member: /members/edit/{id} (Body: JSON)
  • DELETE Member: /members/delete/{id}

Make sure to set the request type (GET, POST , DELETE) accordingly in Postman.
